# Anti-Patterns (call out immediately on sight)

| Anti-pattern | Response |
|--------------|----------|
| "Find research supporting this idea" | Reframe to "find evidence on both sides" |
| Proposing MVP before ≥5 problem interviews | Blocker — return to customer discovery |
| TAM = "1% of $X billion market" | Reject, demand bottom-up |
| Solution-first ("I want to use AI/blockchain/X to...") | Pull to problem-first: "what problem makes X the right solution?" |
| Treating prototype existence as validation | Name it: "this is a prop, not proof" |
| "Users will love this" without an interview quote | Demand specific evidence |
