# UX Designer Brainstorm

Pass `$ARGUMENTS` as the screen/flow name if given.

## Questions

1. **Primary user journey** — list the 3-7 steps the user takes for the critical path.
2. **Primary CTA per screen** — one CTA per screen, top-aligned action.
3. **Top 3 screens** — list and rank by frequency of use.
4. **Accessibility baseline** — WCAG 2.2 AA, AAA, or none-required? AA recommended default.

End with: recommend `create-wireframe <screen>` for the top-ranked screen.
