# Ship + Grow Brainstorm

Cross-reference `shared/references/indie-growth-playbook.md` surfaces table.

## Questions

1. **Which surface fits?** — Product Hunt, HN, X, Reddit, IH, LinkedIn, TikTok/YouTube. Match surface to product type.
2. **Day-1 success** — what number proves the launch worked? (signups, upvotes, comments)
3. **Day-7 success** — what number proves retention? (activation %, return visits)
4. **Day-14 success** — what number triggers ITERATE vs KILL? (paying conversion, NPS, churn)
5. **Feedback channel** — email, Discord, in-app, form? One channel, monitored daily.

End with: recommend `launch-checklist <surface>` next.
